# 사이즈별 비균일 스케일링 전략 보고서

> 작성일: 2026-04-08 | 작성자: planner-architect
> 대상: 승화전사 유니폼 패턴 자동 생성 프로그램 (grader)

## 1. 문제 정의: "비균일 변화"란?

사이즈가 바뀔 때 몸의 모든 부위가 **같은 비율로** 줄어들지 않는다.

```
XL(기준) → XS(타겟) 변화 예시

부위        XL      XS      변화율
어깨폭     45cm    38cm    -15.6%   ← 가장 많이 줄어듦
가슴폭     55cm    47cm    -14.5%
암홀깊이   25cm    22cm    -12.0%
허리폭     52cm    46cm    -11.5%
밑단폭     54cm    48cm    -11.1%
기장       75cm    68cm     -9.3%   ← 가장 적게 줄어듦
```

**비유**: 사진을 축소하면 가로세로가 동일하게 줄지만, 실제 옷은 어깨는 많이 좁아지고 길이는 적게 줄어든다. 마치 **찰흙 인형을 위는 많이 누르고 아래는 적게 누르는** 것과 같다.

이때 디자인에 있는 로고, 번호, 스트라이프 등이 어떻게 변해야 하는가?

## 2. 실제 승화전사 업계의 처리 방식

### 방식 A: 단순 비례 축소 + 수작업 미세 조정 (가장 흔함)

대부분의 승화전사 업체가 사용하는 **현실적 방식**이다.

1. 디자인 전체를 대표 비율(평균값)로 축소한다
2. 패턴 경계에서 미세한 차이는 디자이너가 눈으로 보고 조정한다
3. 인쇄 시 3~5mm의 bleed(여분)을 두므로 작은 오차는 알아서 잘려나간다

**비유**: 복사기로 90%로 축소한 다음, 가위로 모양에 맞게 잘라내는 것.

### 방식 B: 영역 분할 스케일링 (중급 업체)

디자인을 가로/세로 영역으로 나누어 **각각 다른 비율**을 적용한다.

- 예: 상체 영역은 -14%, 하체 영역은 -11%
- Illustrator에서 디자이너가 수작업으로 영역별 변형한다

**비유**: 종이를 가로로 반 잘라서, 위쪽은 많이 줄이고 아래쪽은 적게 줄인 다음 다시 붙이는 것.

### 방식 C: 기준점 매핑 (고급 업체)

패턴의 주요 기준점(어깨점, 겨드랑이점, 허리점 등)을 XL/XS 간에 1:1로 대응시킨다.

- 기준점 사이 영역을 개별적으로 변형한다
- 가장 정확하지만 가장 손이 많이 간다
- 대량 생산보다는 고급 커스텀 유니폼에 사용한다

**비유**: 종이 인형의 관절 부위에서 접어서 각 부위 크기를 개별적으로 조절하는 것.

## 3. 프로그램에서 구현 가능한 4가지 전략

### 전략 1: 단순 비례 + 클리핑 (현실적 MVP)

```
작동 방식:
1. XL/XS 패턴의 전체 가로, 세로 비율을 각각 계산
   (가로 비율 = XS가로 / XL가로, 세로 비율 = XS세로 / XL세로)
2. 디자인을 가로/세로 각각의 비율로 축소
3. 축소된 디자인 위에 XS 패턴 모양을 클리핑 마스크(틀)로 씌움
4. 패턴 밖으로 삐져나온 부분은 잘려나감
```

**비유**: 고무 도장을 가로세로 다르게 눌러 찍고, 쿠키틀로 모양을 찍어내는 것.

| 항목 | 평가 |
|------|------|
| 구현 난이도 | 낮음 (현재 코드에서 자연스러운 확장) |
| 정확도 | 중간 (대부분 충분) |
| CMYK 보존 | 완벽 (기존 show_pdf_page 방식 활용) |
| 예상 기간 | 1~2주 |

### 전략 2: 메시 워프 (격자 변형)

```
작동 방식:
1. 기준 패턴 위에 가로 5 x 세로 8 격자를 만듦
2. 타겟 패턴 위에도 같은 격자를 만듦 (모양이 다름)
3. 격자의 각 칸을 기준 → 타겟으로 변형
4. 디자인의 해당 칸도 동일하게 변형
```

**비유**: 고무판 위에 그림을 그려놓고, 고무판을 부분적으로 늘리거나 줄이는 것.

| 항목 | 평가 |
|------|------|
| 구현 난이도 | 높음 (벡터를 래스터로 변환 필요 가능) |
| 정확도 | 높음 (비균일 변형 정확 반영) |
| CMYK 보존 | 어려움 (래스터화 시 색상 변환 위험) |
| 예상 기간 | 1~2개월 |

### 전략 3: 기준점(앵커) 기반 영역 분할

```
작동 방식:
1. XL/XS 패턴에서 공통 기준점 N개를 정의 (어깨점, 겨드랑이점, 허리점, 밑단 중앙)
2. 기준점 사이 영역을 삼각형/사각형으로 분할
3. 각 영역을 XL → XS 기준점 대응에 맞게 개별 변환
4. 디자인의 해당 영역도 동일하게 변환
```

**비유**: 천 위에 핀을 꽂아놓고, 핀 사이의 천만 줄이거나 늘리는 것.

| 항목 | 평가 |
|------|------|
| 구현 난이도 | 매우 높음 (기준점 자동 인식 어려움) |
| 정확도 | 매우 높음 |
| CMYK 보존 | 어려움 (영역 경계 처리 복잡) |
| 예상 기간 | 2~3개월 |

### 전략 4: 가로/세로 독립 스케일 + 수평 슬라이스

```
작동 방식:
1. 패턴을 수평으로 N개 띠(슬라이스)로 나눔
   예: 어깨 띠, 가슴 띠, 허리 띠, 밑단 띠
2. 각 띠의 XL/XS 가로폭 비율을 개별 계산
   예: 어깨 띠는 -15.6%, 밑단 띠는 -11.1%
3. 디자인의 각 띠를 해당 비율로 가로 축소
4. 세로(기장)는 전체 비율(-9.3%)로 일괄 축소
5. 클리핑 마스크 적용
```

**비유**: 디자인을 가로로 얇게 잘라서 각 조각을 서로 다른 비율로 줄인 다음 다시 쌓는 것.

| 항목 | 평가 |
|------|------|
| 구현 난이도 | 중간 |
| 정확도 | 높음 (가로 방향 비균일 반영) |
| CMYK 보존 | 중간 (슬라이스별 처리 시 주의 필요) |
| 예상 기간 | 2~3주 |

## 5. 승화전사 인쇄의 특수 상황 (오차를 흡수하는 3가지 안전장치)

실제 승화전사 인쇄에는 오차를 자연스럽게 흡수하는 요소들이 있다.

### 안전장치 1: Bleed (여분 인쇄)
- 패턴보다 **3~5mm 크게** 인쇄한다
- 재단할 때 여분이 잘려나가므로, 디자인이 약간 삐져나와도 문제 없다
- **비유**: 색칠할 때 윤곽선 밖으로 약간 삐져나가게 칠하고, 나중에 가위로 오려내는 것

### 안전장치 2: 원단 신축성
- 폴리에스터 원단이 열프레스(고온 인쇄) 시 약간 늘어난다
- 미세한 크기 오차(1~2mm)는 원단이 자연스럽게 흡수한다
- **비유**: 풍선에 그림을 그렸는데 풍선이 약간 부풀어서 그림이 조금 커지는 것

### 안전장치 3: 실무 허용 오차
- 승화전사 업계 표준 허용 오차: **+/- 2~3mm**
- 사이즈 간 최대 비율 차이(15.6% vs 9.3% = 6.3%p)가 만드는 위치 오차는 대부분 이 범위 안에 들어온다
- **비유**: 사진 인화 시 약간의 여백 차이는 눈에 안 띄는 것과 같다

### 오차 계산 예시

```
전략 1(단순 비례)을 사용할 때 발생하는 최대 위치 오차:

평균 축소율: 약 -12.3%
어깨(실제 -15.6%) vs 평균(-12.3%) = 3.3%p 차이
→ 어깨폭 38cm 기준 x 3.3% = 약 1.25mm 오차

이 1.25mm는 bleed(3~5mm) 범위 안에 충분히 들어온다!
```

## 6. 디자인 요소별 처리 전략

| 디자인 요소 | 특성 | 전략 1 결과 | 추가 조치 필요? |
|------------|------|-----------|:---:|
| 배경색/그라데이션 | 전면 채움 | 클리핑으로 패턴에 맞춰짐 | 불필요 |
| 세로 스트라이프 | 전면 패턴 | 비율로 축소, bleed로 커버 | 불필요 |
| 가슴 로고 | 중앙 고정 | 비례 축소, 위치 대략 맞음 | 미세 조정 가능 |
| 등번호 | 중앙 고정, 고정 크기 | 비례 축소 | 별도 처리 예정 |
| 선수명 | 상단 고정 | 비례 축소 | 별도 처리 예정 |
| 패턴 경계 장식 | 경계를 따라감 | 클리핑으로 잘림 | bleed로 커버 |

**핵심**: 대부분의 디자인 요소는 전략 1(단순 비례 + 클리핑)만으로 실무적으로 충분하다. 등번호와 선수명은 어차피 사이즈별로 별도 처리가 필요한 요소이므로 스케일링 정확도와 무관하다.

## 7. Bleed 처리 구현 방안

디자인을 패턴보다 약간 크게 출력하여 재단 여유를 확보하는 방법:

```
방법 A: 스케일 비율에 bleed 반영
  - 목표 크기 = 패턴 크기 + bleed(3~5mm)
  - 스케일 비율 = (패턴폭 + bleed*2) / 기준 패턴폭
  - 클리핑 마스크 없이 출력 → 실제 재단에서 맞춤

방법 B: 클리핑 마스크 확장
  - 패턴 윤곽선을 bleed만큼 offset(확장)하여 클리핑 마스크 생성
  - 패턴보다 약간 큰 영역이 인쇄됨
  - SVG 패턴 경로의 offset 계산이 추가로 필요
```

**추천: 방법 A** (단순하고 현재 코드 구조에 자연스럽게 통합 가능)

## 8. 단계적 구현 로드맵

| Phase | 전략 | 예상 기간 | 정확도 | bleed가 흡수하는 오차 |
|:---:|------|:---:|:---:|:---:|
| **Phase 1** | 단순 비례 + 클리핑 + bleed | 1~2주 | 충분 | +/- 2~3mm 이내 |
| **Phase 2** | 수평 슬라이스 스케일 | 2~3주 | 높음 | +/- 1mm 이내 |
| **Phase 3** | 기준점 매핑 | 1~2개월 | 매우 높음 | 0.5mm 미만 |

---

## 9. 최종 추천

### Phase 1부터 시작하고, 실물 검증 후 판단한다

**이유 3가지:**

1. **업계 현실**: 대부분의 승화전사 업체가 단순 비례 방식을 사용하고 있다. bleed와 원단 신축성이 오차를 흡수하기 때문이다.

2. **오차 범위**: 전략 1의 최대 위치 오차(약 1.25mm)는 업계 허용 오차(+/- 2~3mm) 안에 들어온다. 이론적으로 문제가 없다.

3. **검증 우선**: Phase 1 결과물을 **실제 인쇄**해서 확인하는 것이 가장 확실하다.
   - 오차가 bleed 범위 내면 → Phase 1로 충분 (대부분의 경우 여기서 끝)
   - 특정 디자인(경계 장식이 복잡한 경우 등)에서 문제가 보이면 → Phase 2로 진행
   - Phase 3은 고급 커스텀 유니폼 등 극도의 정밀도가 필요한 경우에만 고려

### 현재 프로그램과의 연결

현재 grader 프로그램의 `pdf_grader.py`는 이미 **show_pdf_page + clip** 방식으로 CMYK 보존 스케일링을 구현하고 있다. Phase 1은 이 기존 코드의 자연스러운 확장이다:

- **기존**: 전체 bounding box 비율로 단일 스케일링
- **Phase 1 추가**: 가로/세로 독립 비율 + bleed 옵션 + SVG 패턴 클리핑 마스크

Phase 2 이상은 `decisions.md`에 기록된 **PDF W 연산자 클리핑 마스크** 기술과 결합하여 구현할 수 있다.
