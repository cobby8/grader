# Grader 프로그램 설치 가이드

유니폼 패턴 자동 생성 프로그램 **Grader** 의 직원용 설치 가이드입니다.
최초 1회만 아래 절차를 진행하면 됩니다. 이후에는 바로 실행할 수 있습니다.

---

## 준비물

- Windows 10 이상 (64비트)
- 인터넷 연결 (설치 시에만 필요)
- 관리자 권한 (3단계에서만 사용)
- Adobe Illustrator (이미 설치되어 있어야 함)

---

## 1단계: Python 설치 (이미 설치돼 있으면 건너뛰기)

Grader 는 내부적으로 Python 을 사용합니다. PC 에 Python 이 없으면 먼저 설치해야 합니다.

### 설치 절차

1. https://www.python.org/downloads/ 접속
2. 노란색 **"Download Python 3.12.x"** 버튼 클릭 (3.9 이상이면 OK)
3. 다운로드된 설치 파일 (`python-3.12.x-amd64.exe`) 더블클릭
4. ⚠️ **중요**: 설치 화면 맨 아래 **"Add python.exe to PATH"** 체크박스를 **반드시 체크**
5. **"Install Now"** 클릭
6. 완료 후 **"Close"** 로 창 닫기

### Python 설치 확인

- 키보드에서 `Win + R` 누르기 → `cmd` 입력 → 엔터
- 검은 창에 다음 명령 입력:
  ```
  python --version
  ```
- `Python 3.12.x` 같은 버전이 표시되면 설치 성공

---

## 2단계: Grader 프로그램 설치

1. 관리자에게 받은 설치 파일을 더블클릭
   - `grader_0.1.0_x64_en-US.msi` 또는
   - `grader_0.1.0_x64-setup.exe`
2. 설치 마법사의 지시에 따라 진행 (기본값 그대로 OK)
3. **"설치"** 클릭 → 완료

설치가 끝나면 바탕화면 또는 시작 메뉴에 **Grader** 아이콘이 생깁니다.

---

## 3단계: Python 엔진 설치 (최초 1회만 - 가장 중요!)

Grader 가 Python 을 사용하기 위한 "엔진"을 설치하는 단계입니다.
이 단계를 건너뛰면 그레이딩이 동작하지 않습니다.

### 절차

1. 설치된 폴더로 이동합니다:
   - **시작 메뉴** → **Grader** 우클릭 → **"파일 위치 열기"**
   - 또는 탐색기에서 `C:\Program Files\Grader\` 로 직접 이동

2. 폴더 안에서 **`setup-python.bat`** 파일을 찾습니다.

3. ⚠️ **중요**: `setup-python.bat` 파일을 **마우스 우클릭** →
   **"관리자 권한으로 실행"** 을 선택하세요.
   (더블클릭하면 권한 부족으로 실패합니다)

4. "이 앱이 디바이스를 변경할 수 있도록 허용하시겠습니까?" 창이 뜨면 **"예"** 클릭

5. 검은 창이 뜨면서 자동으로 설치가 진행됩니다 (3~5분 소요)
   - `[1/4] Python 설치 여부를 확인합니다...`
   - `[2/4] python-engine 폴더를 확인합니다...`
   - `[3/4] 가상 환경(venv)을 생성합니다...`
   - `[4/4] 패키지를 설치합니다...`

6. **"설치 완료!"** 메시지가 나오면 아무 키나 눌러 창을 닫습니다.

---

## 4단계: 프로그램 실행

1. 시작 메뉴 → **Grader** 클릭 또는 바탕화면 아이콘 더블클릭
2. 프로그램이 정상 실행되면 준비 완료입니다!

---

## 문제 해결 (FAQ)

### Q1. "python을 찾을 수 없습니다" 에러

**원인**: 1단계 Python 설치 시 **"Add python.exe to PATH"** 체크를 하지 않은 경우.

**해결**:
1. 제어판 → 프로그램 제거 → Python 3.x 제거
2. 1단계부터 다시 설치 (체크박스 반드시 체크)

---

### Q2. `setup-python.bat` 실행 시 "액세스 거부" / venv 생성 실패

**원인**: 관리자 권한으로 실행하지 않은 경우.

**해결**:
- 파일을 우클릭 → **"관리자 권한으로 실행"** 을 선택하세요.
- 그냥 더블클릭은 권한 부족으로 실패합니다.

---

### Q3. 패키지 설치 실패 (네트워크 에러)

**원인**: 인터넷 연결 문제 또는 회사 프록시/방화벽 차단.

**해결**:
1. 인터넷이 연결돼 있는지 확인
2. 회사 네트워크라면 관리자에게 `pypi.org` 접근 허용 요청
3. 다시 `setup-python.bat` 관리자 권한으로 실행

---

### Q4. 프로그램을 실행해도 그레이딩이 안 됨

**원인**: 3단계 Python 엔진 설치를 건너뛴 경우가 가장 흔함.

**해결**:
- 3단계 다시 실행
- 문제가 계속되면 설치 폴더의 `setup-python.log` 파일을 관리자에게 전달

---

### Q5. Illustrator 관련 에러

**원인**: Adobe Illustrator 가 설치돼 있지 않거나 설치 경로가 비표준.

**해결**:
- Adobe Illustrator 가 `C:\Program Files\Adobe\Adobe Illustrator 20XX\` 경로에 설치돼 있는지 확인
- 설치돼 있지 않다면 Illustrator 를 먼저 설치해야 합니다

## 설치 완료 체크리스트

- [ ] 1단계: `python --version` 명령이 동작

- [ ] 4단계: Grader 실행 시 메인 화면이 정상 표시

모두 체크되면 설치가 완료된 것입니다.
