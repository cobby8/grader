## 📧 [이메일용] 조금 더 풀어쓴 버전

```
제목: [배포 공지] Grader 앱 v1.0.0 설치 안내 드립니다

안녕하세요,

Grader 앱의 첫 공식 버전(v1.0.0)이 배포되었습니다.
아래 안내에 따라 설치해 주세요.

━━━ 설치 순서 ━━━

1) 아래 설명서를 먼저 읽어주세요 (5~10분 소요)
   https://github.com/cobby8/grader/blob/main/INSTALL-GUIDE-STAFF.md

2) 설치 파일 다운로드 (MSI, 4.99 MB)
   https://github.com/cobby8/grader/releases/download/v1.0.0/Grader_1.0.0_x64_en-US.msi

3) 더블클릭 → "추가 정보" → "실행" → 기본값 그대로 설치

━━━ 주요 안내 사항 ━━━

• "Windows에서 PC 보호" 파란 경고는 정상입니다.
  ("추가 정보" 링크 클릭 후 "실행" 버튼 눌러주세요)

• 앞으로 새 버전이 나오면 앱이 자동으로 알려드립니다.
  (설치 파일을 매번 다시 받을 필요 없음)

• Google Drive for Desktop이 설치되어 있어야 합니다.
  (파일 탐색기에서 G: 드라이브가 보여야 함)

━━━ 주요 신규 기능 ━━━

• 양면 유니폼 그레이딩 지원 (U넥 스탠다드)
• SVG 일괄 표준화 기능 (현재 U넥 양면유니폼 전용)
• 자동 업데이트 시스템
• G드라이브 신규 사이즈 자동 감지

문의 사항은 PM에게 연락 주세요.
```

■ 설치 설명서 (필독)
  https://github.com/cobby8/grader/blob/main/INSTALL-GUIDE-STAFF.md

  • MSI 설치 파일 (권장, 4.99 MB)

    https://github.com/cobby8/grader/releases/download/v1.0.0/Grader_1.0.0_x64-setup.exe
  • 전체 Release 페이지
    https://github.com/cobby8/grader/releases/latest

■ 전제 조건
  1. Windows 10 이상
  2. Google Drive for Desktop 설치 (G: 드라이브 마운트 필요)
  3. Python 3.x 설치 (일부 기능용, 권장)

■ 참고 사항
  • Windows SmartScreen "PC 보호" 경고는 정상입니다.
    → "추가 정보" → "실행" 순서로 진행
  • 이후 업데이트는 앱이 자동 감지하여 안내합니다.
  • SVG 표준화 기능은 현재 U넥 양면유니폼 스탠다드만 지원.

■ 문의
  PM
