# Grader 설치 안내 (직원용)

> 대상: Grader 앱을 처음 설치하는 직원
> 예상 소요: 5~10분
> 최종 업데이트: 2026-04-22
> 현재 버전: v1.0.0

---

## 이 가이드가 뭔가요?

Grader 앱을 여러분 PC에 설치하는 방법을 **비개발자도 따라할 수 있게** 풀어 쓴 안내서입니다.
단계별 번호를 보며 순서대로 따라 하시면 됩니다.

**비유하면**:
- Grader 설치 = 스마트폰에 새 앱 설치하기
- 자동 업데이트 = 스마트폰 앱 스토어처럼, 새 버전이 나오면 알아서 "업데이트 하실래요?" 물어봐 줌
- Windows SmartScreen 경고 = "모르는 번호로 온 문자" 같은 것 → 우리 앱이니까 무시하고 진행하면 됨

---

## ⚠️ 설치 전 확인 사항 (전제조건)

아래 조건이 맞는지 먼저 확인하세요.

### 1. Windows 10 이상

- Windows 7/8은 지원하지 않습니다.
- 확인 방법: 바탕화면에서 `내 PC` 오른쪽 클릭 → `속성` → 화면 우측 "Windows 사양"에서 **Windows 10** 또는 **Windows 11** 표시 확인

### 2. Google Drive for Desktop 설치 (G: 드라이브 마운트)

- Grader는 디자인 SVG 패턴을 **Google 공유 드라이브**에서 읽어옵니다.
- 확인 방법: 파일 탐색기를 열어서 **`G:` 드라이브**가 보이는지 확인 (보통 `공유 드라이브` 이름으로 표시됨)
- 없다면: [Google Drive for Desktop 다운로드](https://www.google.com/intl/ko/drive/download/)에서 설치 후, 회사 Google 계정으로 로그인하세요.

### 3. Python 3.x 설치 (권장, 필수는 아님)

- 내부 엔진이 Python으로 작동합니다. 대부분의 PC에는 이미 설치돼 있을 수 있어요.
- 확인 방법: `Win + R` → `cmd` 입력 → 검은 창에 `python --version` 입력 → `Python 3.x.x`라고 나오면 OK
- 없으면: [python.org에서 Python 3.11+ 다운로드](https://www.python.org/downloads/) 후 설치
  - 설치 마법사에서 **"Add python.exe to PATH"** 체크박스를 반드시 켜주세요

> 💡 **Python이 없어도 대부분의 기능은 동작**합니다. 단, 일부 고급 기능(SVG 표준화 등)은 Python이 필요해요.

## 설치 절차

### [1] 설치 파일 다운로드

1. 브라우저(크롬, 엣지 등)를 엽니다.
2. 주소창에 아래 URL을 붙여넣고 엔터:
   ```
   https://github.com/cobby8/grader/releases/latest
   ```
3. 페이지 아래로 스크롤하면 **"Assets"** 항목에 파일 목록이 보입니다.
4. **`Grader_1.0.0_x64_en-US.msi`** 파일을 클릭해서 다운로드.

<!-- SCREENSHOT: GitHub Releases 페이지의 Assets 섹션과 MSI 파일 위치 -->

> 💡 파일명이 조금 다를 수 있어요 (버전 숫자 부분 `1.0.0` 자리가 바뀜). `.msi` 확장자가 붙은 파일을 받으시면 됩니다.

---

### [2] "Windows에서 PC 보호" 경고 넘기기

**왜 이런 경고가 뜨나요?**

- Grader는 공식 코드 사인 인증서(연 수십만 원)를 구매하지 않았습니다.
- 그래서 Windows는 "모르는 프로그램이네?"라고 걱정해주는 것일 뿐, **실제로 위험한 건 아닙니다**.
- 카톡/크롬은 대형 회사라서 인증서를 샀지만, 우리 같은 사내 소규모 앱은 건너뛰어도 OK.

**경고 넘기는 방법**:

1. 다운로드한 `.msi` 파일을 더블클릭합니다.
2. 파란색 창이 뜨면서 **"Windows에서 PC 보호"** 문구 표시.
3. 창 중앙의 **"추가 정보"** 링크를 **클릭**합니다.
4. 아래에 **"실행"** 버튼이 새로 나타납니다. 클릭!

<!-- SCREENSHOT: Windows SmartScreen "추가 정보" → "실행" 2단계 -->

> ⚠️ **주의**: 다른 경고(예: 빨간색 "잠재적 보안 위협" 등)가 뜨면 실행하지 마시고 PM에게 문의하세요. 파란색 SmartScreen 경고만 정상입니다.

---

### [3] 설치 마법사 따라가기

설치 프로그램 창이 뜹니다. **기본값 그대로** 진행하면 됩니다.

1. "Next" 클릭
2. 설치 경로 (기본값: `C:\Program Files\Grader`) → "Next"
3. "Install" 클릭
4. **"사용자 계정 컨트롤(UAC)"** 창이 뜨면 "예" 클릭 (관리자 권한 요청)
5. 설치 진행 (1~2분)
6. "Finish" 클릭

<!-- SCREENSHOT: 설치 마법사 각 단계 -->

---

### [4] 앱 실행

1. Windows 시작 메뉴를 엽니다 (`Win` 키 누르기)
2. 검색창에 **"Grader"** 입력
3. 맨 위에 Grader 앱이 나타나면 클릭해서 실행

<!-- SCREENSHOT: 시작 메뉴에서 Grader 검색 -->

> 💡 자주 쓰실 거라면 시작 메뉴에 고정하거나, 작업 표시줄에 고정해두시면 편해요.

## 초기 설정 (최초 1회만)

앱을 처음 실행했을 때 해두어야 할 설정입니다.

### [1] Drive 패턴 루트 폴더 경로 설정

**왜**: Grader가 G드라이브 어디에 있는 SVG 파일을 읽어올지 알려주는 단계.

1. 좌측 사이드바에서 **"설정"** 클릭
2. **"Drive 연동"** 섹션 찾기
3. **"패턴 루트 폴더"** 입력란에 아래 경로 붙여넣기:
   ```
   G:\공유 드라이브\디자인\00. 2026 커스텀용 패턴 SVG
   ```
   > ⚠️ 경로는 PM이 지정한 현재 사용 경로로 입력하세요. 연도별/부서별로 달라질 수 있습니다.
4. **"Drive 연동 활성화"** 토글을 **ON**으로
5. "저장" 클릭

<!-- SCREENSHOT: 설정 페이지의 Drive 연동 섹션 -->

---

### [2] 패턴 관리 페이지 진입 (자동 스캔)

1. 좌측 사이드바에서 **"패턴 관리"** 클릭
2. 처음 진입하면 자동으로 G드라이브 폴더를 스캔합니다 (10~30초)
3. 스캔이 끝나면 패턴 카드 목록이 표시됩니다

<!-- SCREENSHOT: 패턴 관리 페이지 스캔 완료 상태 -->

> 💡 이후에는 앱 켤 때마다 60초 쿨다운 후 자동으로 최신 상태를 가져옵니다. 강제로 새로 고치려면 "수동 스캔" 버튼을 누르세요.

## 자동 업데이트 사용법

Grader는 앞으로 **앱을 켤 때마다** 새 버전을 확인합니다.

### 새 버전 있을 때

1. 앱을 실행하면 잠시 후 **"새 버전 v1.x.x이 있습니다"** 모달 창이 뜸
2. 변경사항 내용을 읽어본 후:
   - **"업데이트"** 클릭 → 다운로드(진행률 표시) → 자동 설치 → 앱 재시작
   - **"나중에"** 클릭 → 팝업만 닫힘 (다음 실행 시 다시 물어봄)

<!-- SCREENSHOT: UpdateModal 업데이트 안내 팝업 -->

> 💡 업데이트는 보통 1~3분 걸립니다. 중간에 PC 종료/인터넷 끊김이 있어도 기존 버전이 유지되므로 안전합니다.

### 수동으로 업데이트 확인

"나중에"로 닫았거나, 중요한 수정이 있다고 공지받았을 때 수동으로 확인할 수 있습니다.

1. 좌측 사이드바 **"설정"** 클릭
2. 맨 아래 **"버전 정보"** 섹션으로 스크롤
3. **"[업데이트 확인]"** 버튼 클릭
4. 새 버전이 있으면 업데이트 모달이 뜨고, 없으면 "최신 버전입니다" 메시지

## 현재 기능 제한 (v1.0.0 기준)

아래 항목은 **첫 릴리스의 범위 제한**이므로, 추후 업데이트로 점차 확장될 예정입니다.

### SVG 표준화: U넥 양면유니폼 스탠다드만 지원

- 패턴 카드 `⋮` 더보기 메뉴의 **[📐 SVG 표준화]** 기능은 현재 **U넥 양면유니폼 스탠다드 폴더에서만** 동작합니다.
- ⚠️ **슬림, V넥, 하의 등 다른 폴더**에서 이 버튼을 누르면 "변환 실패" 메시지가 나타나며 **파일은 수정되지 않습니다** (안전함).
- 슬림/V넥/하의 지원은 **추후 업데이트 예정**이며, PM이 별도로 공지합니다.

### Drive 연동: G: 드라이브 전용

- 현재는 `G:\` 드라이브에 Google 공유 드라이브가 마운트돼 있어야 합니다.
- 다른 드라이브 문자(D:, E: 등)를 쓰는 경우 PM에게 문의하세요.

## 자주 묻는 질문 (FAQ)

### Q1. 설치했는데 `Python이 필요합니다` 에러가 떠요.

**A**: [전제조건 3](#3-python-3x-설치-권장-필수는-아님) 참고하여 Python 3.11+ 설치하세요. 설치 중 **"Add python.exe to PATH"** 체크 잊지 마세요.

### Q2. G드라이브가 보이지 않아요.

**A**: Google Drive for Desktop이 꺼져 있을 수 있습니다.
1. 작업 표시줄 우측 시스템 트레이에서 Drive 아이콘 (삼각형+Drive 로고) 확인
2. 없으면 시작 메뉴에서 "Google Drive" 검색 후 실행
3. 회사 Google 계정으로 로그인

### Q3. 앱이 느려요 / 응답이 없어요.

**A**: 처음 켤 때 G드라이브 스캔이 오래 걸릴 수 있어요 (SVG 파일 수에 비례).
1분 넘게 응답 없으면 앱을 종료 후 다시 켜보세요. 계속 그러면 PM에게 문의.

### Q4. 업데이트 중에 실수로 PC를 껐어요. 앱이 망가졌나요?

**A**: 안전합니다. Tauri 자동 업데이터는 **새 파일이 완전히 준비된 후에만** 기존 파일을 교체합니다.
중간에 끊기면 기존 버전이 그대로 유지되므로, 그냥 다시 앱 켜고 업데이트 팝업에서 "업데이트" 다시 누르세요.

### Q5. "Windows에서 PC 보호" 경고가 무서워요. 정말 안전한가요?

**A**: 네, 안전합니다.
- 우리 앱은 사내에서 만들어 GitHub에 올린 공식 빌드입니다.
- "코드 사인 인증서"라는 유료 인증만 없을 뿐, 내용은 정상입니다.
- **단, `https://github.com/cobby8/grader/releases/latest`에서 직접 받은 파일만** 신뢰하세요. 이메일 첨부나 이상한 링크로 받은 파일은 실행하지 마세요.

### Q6. 업데이트 팝업이 안 떠요. 최신 버전인지 어떻게 확인하죠?

**A**:
1. 설정 → 버전 정보 → 현재 버전 확인 (예: "v1.0.0")
2. 같은 페이지에서 **[업데이트 확인]** 수동 버튼 클릭
3. 최신이 맞으면 "최신 버전입니다" 메시지 표시

### Q7. 업데이트가 회사 네트워크에서 차단되는 것 같아요.

**A**: 회사망이 `github.com`을 차단했을 수 있습니다.
- 일시적으로 모바일 핫스팟에서 업데이트 시도
- 지속적으로 안 되면 PM에게 문의 (IT팀 방화벽 예외 요청 필요)

문의 시 다음 정보를 함께 알려주시면 빠른 해결에 도움이 됩니다:
