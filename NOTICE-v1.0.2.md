# v1.0.2 핫픽스 공지 (직원 전달용)

> 카카오톡 단체방 / 이메일 / 사내 게시판에 그대로 복붙하세요.
> v1.0.0 / v1.0.1 사용자에게 **반드시 전달**해 주세요. 자동 업데이트가 안 뜨는 분이 계실 수 있습니다.

---

## 🚨 핵심 한 줄

> **신규 PC에서 첫 실행 시 "저장 실패: 지정된 경로를 찾을 수 없습니다 (os error 3)" 가 뜨던 결함이 v1.0.2에서 해결됐습니다.**
> 이 결함을 겪고 계신 분은 **자동 업데이트가 안 보일 수 있으므로 v1.0.2 설치 파일을 직접 받아서 재설치**해 주세요.

## 📧 [이메일용] 조금 더 풀어쓴 버전

```
제목: [핫픽스] Grader v1.0.2 안내 — 신규 PC 첫 실행 결함 해결

안녕하세요,

Grader 앱 v1.0.2 핫픽스가 배포되었습니다.
v1.0.1 까지에서 신규 PC 처음 사용하실 때 "저장 실패: 지정된 경로를
찾을 수 없습니다(os error 3)" 메시지로 진행이 막히던 결함을 고친 버전입니다.

━━━ 본인 상황별 안내 ━━━

【1】 이미 Grader 앱이 정상 동작 중이신 분
   → 다음 앱 실행 시 자동 업데이트 안내 모달이 뜹니다.
     "지금 업데이트" 버튼 눌러주시면 됩니다.
   → 만약 모달이 안 뜨면 [설정] 메뉴 > [업데이트 확인] 버튼을 눌러주세요.

【2】 신규로 설치하셨다가 첫 실행에서 막혀 계신 분
   → 자동 업데이트가 동작하지 않을 수 있으므로
     아래 링크에서 v1.0.2 설치 파일을 직접 받아 재설치해 주세요.
     기존 설치본 위에 그대로 덮어쓰면 됩니다.

【3】 아직 설치를 안 하신 분
   → 처음부터 v1.0.2를 설치하시면 됩니다.

━━━ 다운로드 링크 ━━━

• MSI 설치 파일 (권장)
  https://github.com/cobby8/grader/releases/download/v1.0.2/Grader_1.0.2_x64_en-US.msi

• EXE 설치 파일
  https://github.com/cobby8/grader/releases/download/v1.0.2/Grader_1.0.2_x64-setup.exe

• 전체 변경사항
  https://github.com/cobby8/grader/releases/tag/v1.0.2

━━━ 설치 시 안내 ━━━

• "Windows에서 PC 보호" 파란 경고는 정상입니다.
  ("추가 정보" → "실행" 순서로 진행)

• 첫 실행 시 정상이면 좌하단에 "v1.0.2"가 표시됩니다.

문의는 PM에게 주세요.
```
